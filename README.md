# Sentiment Analysis Model Benchmark

## Overview
This project compares multiple machine learning models for sentiment classification on movie reviews.
The goal is to analyze how different algorithms perform on the same NLP task.

## Dataset
IMDB Movie Review Dataset

- 50,000 labeled reviews
- 25,000 positive
- 25,000 negative

## Current Pipeline

Text Data
→ TF-IDF Vectorization
→ Logistic Regression
→ Evaluation

## Current Results

| Model | Accuracy | F1 Score |
|---|---|---|
| Logistic Regression + TF-IDF | 89.5% | 0.90 |

## Concepts Covered

- Supervised Learning
- Train/Test Split
